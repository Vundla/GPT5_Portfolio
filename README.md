# GPT5 Portfolio Monorepo

This is the monorepo for the GPT5 Portfolio project.

## Structure

- `apps/portfolio-site`: Solid.js frontend
- `apps/api-gateway`: ASP.NET Core 10 API Gateway
- `apps/ai-playground`: FastAPI AI Engine

## Getting Started

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Start development environment:
   ```bash
   pnpm dev
   ```
